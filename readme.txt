QUASI88 0.6.1 for PSP relase 1
簡易説明書

QUASI88 0.6.1をベースにPSPに再度移植した物です。
PSPSDKを使用することで非GPL化しています。
（Q88Pのソースは含んでいません)

Rトリガー　メニュー
Lトリガー　ソフトウェアキーボード

ソフトウェアキーボード上
　CIRCLE ->　押す
   CROSS -> ロック/アンロック
  SQUARE -> RETURN
TRIANGLE -> SHIFT

SAVE SETTINGで設定を保存可能です。
保存された設定は次回起動時に読み込まれます。

ライセンスはQUASI88に準拠しています。

また、PSP版は特殊な方法によりプログラムを起動していますので、
このプログラムをオーバークロック状態を含む、あらゆる状態において、
使用したことにより生じる一切の現象に対して
保証されませんのでご注意ください。
